Indroduction

C is popular high level language that is used in system admisitration, embedded systems software programming, operating systems, and in networking

Basic Structure of C Program

The basic structure of a C program is very simple and flexible. It consists of the following parts:

  • Preprocessor directive

    The first line is a preprocessor directive to include the header file stdio.h. Preprocessor directive gives instructions to C preprocessor. Preprocessor is a program that modifies C source program before before compilation. The preprocessor directives starts with the # symbol

  • Main() function

    The second line is a main function. The main() function is the place where the execution of the C program starts. When a program executes, the control enters main() function and starts executing it's statements

  • Program body (C statements)

    The statements of the programs are written in curly braces. The braces are also known as delimeters. These statements are objectively known as the body of the program. Every statement in C is terminated with a semi colon (;). The compiler generates an error if any statement is not terminated bya semi colon.

    #include < stdio.h >
    void main()
    {
    printf("Hello World");
    }

Variable Declaration

  • Variable

    A variable is a name memory location or memory cell. It is used to store a programs input data and It's computational results during execution. The values of a variable may change during the execution of a program. However, the name of the variables connot be changed

  • Variable declaration

    The process of specifying the variable name and It's type is called variable declaration

    Syntax

    The syntax of declaring variable is as follows

    data_type variable_name

    Example

    int marks;
    float average;
    char grade;

  • Variable initialization

    The process of assigning a value to a variable at the time of declaration is known as variable initialization. The equal sign is used to initialize a variable. Variable name is written on the left side and the value is written on the right side of equal sign

    Syntax

    type_name variable = value;

    Example

    int n = 100;
    int x = 50;
    float average = 50.73;

Constants

  • Constants

    A constant is a quantity that cannot be changed during program execution

    • C language provides the following type of constants:

  • Integer Constants

    Integer constants are used to store integer values

    Example:

    40

    74

    2

  • Floating Point Constants

    Floating point constants are used to store floating point values.

    Example:

    35.5

    63.567

    1.7834

  • Character Constants

    Character Constants are used to store characters. Anythin written within a single quote is a character.

    Eample:

    'C'

    'Hello'

    'N'

  • String Constants

    String Constants are used to store strings. Anything written within double quotes is a string.

    "Good"

    "12345"

    "USA"

Data Types

  • Data Types

    The data type specifies the type of data that can be stored in a variable

    • C language provide the following standard data types

  • Integer Data

    Integer data consists of numbers without floating points

    Example:

    int b = 4;
    int k = 31;
    int n = 41;

  • Floating Point Data

    Floating point data are numeric values that consists of floating point

    Example:

    float x = 47.980;
    float z = 31.002;
    float a = 5.23;
